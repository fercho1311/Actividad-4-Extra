El programa solicita los datos del usuario para realizar una cotización de seguro, pidiendo su nombre, edad, si está casado, la edad del conyuge, si tiene hijos, la cantidad de hijos, si tiene propiedades, la cantidad de propiedades y su salario, de acuerdo a los datos se realizaran algunos recargos dependiendo de la información ingresada.


Las partes a considerar son que el asegurado tiene que ser mayor de edad al igual que el conyuge, tienen que ingresarse el tipo de datos correctamente ya sean números o letras.

Los problemas a mejorar:

para cada solicitud de datos se repita la pregunta hasta que sea la respuesta correcta, mostrar cada recargo que se le realizara para que el cliente tenga claro esos datos, mostrar el dato de recargos más el precio base del seguro y luego el total, tener más variedad de seguros.
